Additional Information

Purpose

The primary use of this data is to provide information about Activities and Attractions identified at specific civic address locations in B.C.

Object Description

Object Name: WHSE_IMAGERY_AND_BASE_MAPS.GSR_HELLOBC_ATTRACTIONS_SVW

Short Name: HLLBC_ATT
Comments: GSR_HELLOBC_ATTRACTIONS_SVW is a spatially enabled layer comprising HelloBC non-accommodation products listing, e.g., transportation, activities and attractions, events, etc.

Column Name Short Name Data Type Data Precision Comments
CUSTODIAN_ORG_DESCRIPTION CUST_ORG VARCHAR2 200 CUSTODIAN_ORG_DESCRIPTION contains the name or description of the custodial organization e.g. Ministry of Agriculture.
BUSINESS_CATEGORY_CLASS BUS_CAT_CL VARCHAR2 1000 BUSINESS_CATEGORY_CLASS is the designation that the business belongs to. E.g. accommodationFoodServices.accommodations or educationalServices.multiLevelSchool
BUSINESS_CATEGORY_DESCRIPTION BUS_CAT_DS VARCHAR2 1000 BUSINESS_CATEGORY_DESCRIPTION is the description that applies to its corresponding BUSINESS_CATEGORY_CLASS. E.g. Tourist accommodations for class accommodationFoodServices.accommodations.
OCCUPANT_TYPE_DESCRIPTION OCCPNT_TYP VARCHAR2 500 OCCUPANT_TYPE_DESCRIPTION contains the description of the occupant type e.g. Hospital
SOURCE_DATA_ID SRCDATA_ID VARCHAR2 20 SOURCE_DATA_ID contains the identifier from Source System (e.g. the internal hospital id of the source system might be A1234)
OCCUPANT_NAME NAME VARCHAR2 500 OCCUPANT_NAME is a business name that can identify the occupant who provides the BC Government or BC Government related services to public. e.g., Burnaby General Hospital, Golden Food Bank
DESCRIPTION DESCRIPTN VARCHAR2 4000 DESCRIPTION describes the Occupant in more detail. e.g., Burnaby General Hospital (BGH) that is managed by Fraser Health Authority provides a wide range of integrated health care services from acute care hospitals to community-based residential, home health, mental health and public health.
PHYSICAL_ADDRESS ADDRESS VARCHAR2 1000 PHYSICAL_ADDRESS contains the civic or non-civic address as a single string, structured according to the specification of the Physical Address and Geocoding Standard, e.g., 420 GORGE RD E, VICTORIA, BC.
ALIAS_ADDRESS ALIAS_ADDR VARCHAR2 1000 ALIAS_ADDRESS contains an address string, not a parsed address. It is the address that will be displayed for presentation purposes. Note that the locality of this address may differ from the locality returned by the DataBC Physical Address Geocoder. E.g. 32900 Marshall Road, Abbotsford, BC
STREET_ADDRESS ST_ADDRESS VARCHAR2 200 STREET ADDRESS is a free form expression of the site descriptor (e.g. unit), civic building number / street / street indicator portion of an address, e.g., Unit 1, 123 Main Street East
POSTAL_CODE POSTAL_CD VARCHAR2 10 POSTAL CODE is the Canadian Postal code value associated with the physical address, e.g., V9Z 2K1
LOCALITY LOCALITY VARCHAR2 100 Locality is the name of the municipality, community, Indian reservation, subdivision, regional district, aboriginal lands, or natural feature the occupant site is located in, e.g., Saanich
CONTACT_PHONE CONT_PHONE VARCHAR2 50 CONTACT_PHONE contains the "general office" phone number of the Occupant, e.g., 250-555-1234
CONTACT_EMAIL CONT_EMAIL VARCHAR2 100 CONTACT_EMAIL contains the "general office" email address of the Occupant, e.g., info@organization.com
CONTACT_FAX CONT_FAX VARCHAR2 50 CONTACT_FAX contains the "general office" fax number of the Occupant, e.g., 250-555-1234
WEBSITE_URL WEBSITE VARCHAR2 500 WEBSITE_URL contains the full URL link to the Home page of the Occupant Website, e.g., www.occupant.com/occupant.html
IMAGE_URL IMAGE_URL VARCHAR2 500 IMAGE_URL contains a full URL link to a picture of the Occupant's Location, e.g., http://www.occupant.com/image.jpg
LATITUDE LATITUDE NUMBER 9 LATITUDE is the geographic coordinate, in decimal degrees (dd.dddddd), of the location of the feature as measured from the equator, e.g., 55.323653
LONGITUDE LONGITUDE NUMBER 10 LONGITUDE is the geographic coordinate, in decimal degrees (ddd.dddddd), of the location of the feature as measured from the prime meridian, e.g., -123.093544
KEYWORDS KEYWORDS VARCHAR2 1000 KEYWORDS contains text strings supplied by the Custodian, to be used for search/query purposes. Keywords are separated by the ; delimiter, e.g., courts;provincial;courts of law
DATE_UPDATED DT_UPDATE DATE 7 DATE_UPDATED contains the date that the Occupant data was updated in the Occupant structure (system generated)
SITE_GEOCODED_IND GEOCD_IND VARCHAR2 1 SITE_GEOCODED_IND contains a Flag/indicator (Y/N) that the Occupant Physical Address has been geo-coded by the DataBC Address Geocoder and the results provide a valid site address, e.g., Y, N
FACEBOOK_ACCOUNT FACEBOOK VARCHAR2 2083 FACEBOOK ACCOUNT provides the facbook link of the Event
PRODUCT_ID PRODUCT_ID VARCHAR2 20 PRODUCT_ID is an Identifer from the Source System - Product_AggressoID.
SERVICE_NAME PROD_GRP VARCHAR2 500 SERVICE_NAME stores the name of services provided by the occupant. i.e., Activity / Attraction OR Transportation OR Golf Courses OR Skiing & Snowboarding
TOURISM_AREA TRSM_AREA VARCHAR2 200 TOURISM_AREA is a geographic area which identifies the location of attractions and businesses. e.g., Vancouver Island; Vancouver, Coast & Mountains; Kootenay Rockies
TWITTER_ACCOUNT TWITTER VARCHAR2 2083 TWITTER ACCOUNT is the Twitter link for the business.
SHAPE SHAPE SDO_GEOMETRY 64 SHAPE is the column used to reference the spatial coordinates defining the feature.
SEQUENCE_ID SEQ_ID NUMBER 10
SE_ANNO_CAD_DATA BLOB 4000 SE ANNO CAD DATA is a binary column used by spatial tools to store annotation, curve features and CAD data when using the SDO_GEOMETRY storage data type.
